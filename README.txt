## Projeto de Agricultura 4.0 usando MQTT

## Arquivos:
- sensor_temperatura.py
- sensor_umidade.py
- monitor.py
- irrigador.py

Como executar:

1. Instalar dependência:
pip install -r requirements.txt

2. Rodar em terminais separados:
python sensor_temperatura.py
python sensor_umidade.py
python monitor.py
python irrigador.py
